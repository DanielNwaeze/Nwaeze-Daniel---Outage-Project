What This Project Does

This projects implements accurate modelling with GIS techniques to identify unplanned High Impact Low Probability (HIPL) outages on IKEDC Distribution Network, which aids resilience and outage mitigation.
It also visualises outages using the CNN Logs on OpenStreetMap.

Outage Prediction System

Predict outages, both planned and unplanned, for High Impact Low Probability (HILP) events to aid resilience and outage mitigation, leveraging advanced analytics for accurate, proactive forecasting.

Outage Classification System

Classify fault descriptions from fault logs into either: Breaker, Feeder, High-Voltage (Distribution) System, Human, Customer Notification, Planned, Substation, Transmission, and Transformer faults

System Operations and Weather Data

Extract Nigeria's system operations and weather data to predict outages. Specify dates and access detailed weather info to enhance predictive models, improving infrastructure and resilience.

Outage Mapping System

Track power outages over time, visualize affected areas with heatmaps, identify geographic clusters, and filter by location. Helps to pinpoint issues, target interventions, and improve power reliability across Nigeria.

Project Overview

‚ö° Challenge

Power outages, both planned and unplanned, pose significant challenges to infrastructure resilience and reliability, especially in regions prone to High Impact Low Probability (HILP) events. These outages can disrupt daily life, economic activities, and essential services, highlighting the need for effective prediction and mitigation strategies.

üîç Scope of the Project

This project aims to enhance the prediction, classification, and mapping of power outages to improve resilience and mitigation efforts. It leverages advanced machine learning and geospatial data solutions to provide accurate, actionable insights that help stakeholders proactively address and manage power outages.

üõ†Ô∏è Methodology

Data Integration: Previous day's outage data, weather information, and system operations data are integrated using FastAPI for efficient data handling and processing. This backend technology facilitates the collection and management of diverse data sources.

Classification: Power outages are classified into detailed categories using machine learning algorithms like Random Forest and Logistic Regression. These models, implemented in the backend, analyze data to quickly identify and categorize fault types, improving maintenance and repair efforts.

Weather and Operations Data: System operations and weather data are extracted and integrated using FastAPI, ensuring smooth data retrieval and processing. Frontend technologies like HTML, CSS, and JavaScript enable users to specify dates and access detailed weather and operational statistics. This integration reveals patterns that enhance predictive models.

Mapping and Visualization: Geographic Information Systems (GIS) technologies like OpenStreetMap, Folium, and GeoPandas are used to track power outages over time, visualize affected areas with heatmaps, identify geographic clusters, and filter by location. These tools are crucial for creating interactive and informative maps that help pinpoint issues and target interventions.

üìä Results

The integration of diverse data sources and advanced analytics has yielded reliable predictions of potential outages, enabling proactive measures to mitigate disruptions. Detailed classification of outages helps quickly identify and resolve issues, minimizing downtime and improving service reliability. By correlating outage data with system operations and weather information, the project provides actionable insights for efficient resource allocation and planning.
